Les feuilles de styles liées

L'utilisation des directives de styles incorporées au document HTML ne présente pas beaucoup d'intérêt.

Par contre, l'ensemble des directives données peut être groupé dans un document qui ne sert qu'à contenir des directives de styles: on parle alors d'une feuille de styles. Les feuilles de styles liées sont des outils extrêmement puissants.

Il suffit d'établir un lien entre un document HTML et une feuille de style pour que toutes les directives soient appliquées à la page HTML.

L'établissement d'une feuille de style est très simple: il suffit de confectionner un document qui contient exclusivement:

  • une balise <style>
  • un certain nombre de directives de style
  • une balise </style>

Un tel document peut être rédigé à l'aide de n'importe quel logiciel capable d'enregistrer un document en mode texte.

La feuille de style se reconnaît à son extension "CSS".

La feuille de style peut également s'établir à l'aide du logiciel Frontpage.

Fabriquer une feuille de style avec Frontpage

Dans le menu "Fichier", sélectionner la commande "Nouveau" et l'option "Page".

Dans la boîte de dialogue, sélectionner l'onglet "Feuilles de style".

Pour fabriquer une nouvelle feuille de style, choisir "Feuille de style Normal".

Sur cette page, il suffit de "coller" les directives de styles "coupées" du document de travail. Attention: seules les directives de style doivent être transférées; les balises <style> et </style> sont intégrées automatiquement dans la feuille de style.

Enregistrer ensuite cette feuille de style sous n'importe quel nom, avec l'extension CSS.

On peut ensuite fermer ce document (Fichier > Fermer).

Lier une feuille de style avec Frontpage

Le document de travail est maintenant privé de tous les styles qui ont été définis. Nous allons établir la liaison avec la feuille de style.

En mode "Normal", sélectionner la commande "Liens de feuilles de style..." dans le menu "Format".

Dans la boîte de dialogue suivante, cliquer sur "Ajouter" et rechercher la feuille de style enregistrée précédemment.

Dès que le lien est établi, la présentation du document est mise à jour.

Dans le document HTML, on trouve le lien avec la feuille de style.

Modifie quelques directives de la feuille de style et observe les effets de cette modification sur le document de travail.

Lie cette feuille de style à un deuxième document quelconque. Observe l'effet.

Lie cette feuille de style à plusieurs pages d'un même site web de manière à lui conférer une certaine unité graphique.
